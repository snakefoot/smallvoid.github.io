Windows 2000 / XP has the ability to make any window transparent, which can be used to turn entire applications transparent which can be useful if having an application turned on all the time, but shouldn't cover the other windows.

- [Glass2k](http://www.chime.tv/products/glass2k.shtml)
- [Trans2k 2.2](http://smallvoid.orgfree.com/?file=trans2k2.2.zip "545 Studios")
- [Vitrite](http://home.insightbb.com/~ryanvm/tinyutilities/vitrite/)
- [TransBar](http://www.home.no/aksoftware/)
- [PowerMenu](http://www.veridicus.com/tummy/programming/powermenu/)
